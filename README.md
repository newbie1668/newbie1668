## Product Principles

- Start with the customer problem, then make the smallest useful thing real.
- Treat prototypes as decision tools, not theater.
- Make experiments measurable before engineering effort gets expensive.
- Use AI tooling to compress the build loop, while keeping product judgment human.
- Write enough documentation that someone else can pick up the work.
